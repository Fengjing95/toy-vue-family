## description
使用 vite + vue3 实现的toy 版 vue-router 和 vuex, 以后尝试自己实现 vue3 替换官方版

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
